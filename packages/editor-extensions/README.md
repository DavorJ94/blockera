# Extensions

extensions package.

## Installation

Install the module:

```bash
npm install @blockera/editor-extensions --save
```

## Usage

```js
import { registerBlockExtension } from '@blockera/editor-extensions';

registerBlockExtension(extensionName, extensionData);
```
