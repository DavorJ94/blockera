# Extensions

Publisher extensions package.

## Installation

Install the module:

```bash
npm install @publisher/extensions --save
```

## Usage

```js
import { registerBlockExtension } from '@publisher/extensions';

registerBlockExtension(extensionName, extensionData);
```
